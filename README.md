## Sobre o projeto

**Hollow Knight 3: Pablo — Edição N64** é uma página web criada como projeto acadêmico para demonstrar conceitos de desenvolvimento front-end utilizando apenas tecnologias base da web.

A proposta foi criar uma página com identidade visual forte, navegação simples e recursos interativos que deixassem a experiência mais divertida do que um site estático tradicional.

Tudo foi construído separando estrutura (**HTML**), estilo (**CSS**) e comportamento (**JavaScript**).

---

## Recursos implementados

### Estrutura
- Header com identidade visual
- Navegação por âncoras
- Organização semântica com:
  - `header`
  - `nav`
  - `main`
  - `section`
  - `footer`

### Visual
- Tema retrô inspirado em Nintendo 64
- Scanlines estilo CRT
- Sombras pixeladas
- Neon glow
- Responsividade para desktop e mobile

### Interatividade
- Galeria com slideshow
- Navegação por teclado
- Lightbox para ampliar imagens
- Trilha sonora com play/pause
- Trailer incorporado via YouTube
- Barra de estoque animada
- Simulação de compra
- Formulário com validação
- Easter Egg do Konami Code

### Acessibilidade
- `aria-label`
- `role`
- Navegação por teclado
- `prefers-reduced-motion`
- Textos alternativos em imagens
